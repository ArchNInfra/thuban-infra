# Thuban – AWS Infra Baseline

Thuban is the foundational AWS infrastructure baseline using Terraform.

## Scope
Minimal AWS infrastructure baseline using Terraform:
- VPC (public/private subnets)
- IAM roles
- ECR registry
- ECS cluster (compute foundation)
- Remote state backend

## Stack
- Terraform
- AWS: VPC / IAM / ECS / ECR
- Backend: S3 + DynamoDB (state locking)
